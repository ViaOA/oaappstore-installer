
Instructions for creating OS specific installer for OAInstaller by using JDK21 JPackage command.

Steps to follow to build OAAppStore Java App:

1: build OAInstaller.jar
    run mvn install 
        creates jar target/OAInstaller-1.0.0.jar

    
NEXT:  create OS specific installer in jpackage folder, subdirectory for the OS and readme.txt 

=> jpackage\windows\readme.txt
